# Agent 文档导航

这些文档承接 `AGENTS.md` 路由出的专项规则。只在任务触发时读取对应文档，避免一次性加载全部细则。

| 文档 | 用途 |
| --- | --- |
| `search-and-evidence.md` | 联网检索、网页抓取、官方文档、API/SDK/库用法、Deep Research、强时效或高风险事实核验 |
| `local-environment.md` | 本地 Shell、路径、文件搜索、环境变量、Homebrew、Ghostty、终端配置 |
| `project-workflow.md` | 项目命令、包管理器、Python/Node、Git、构建测试、安全边界 |
| `requirements.md` | 产品需求和项目背景 |
| `mvp-validation.md` | MVP 验证相关记录 |
| `phase-2-plan.md` | 第二阶段后续动作任务计划 |
| `git-conventions.md` | Git 提交粒度、提交信息和推送规范 |
